## how to use
1. Open the *.dart file
2. Click menu -> code -> JsonToDartClass.
3. Input json and click the ok button.
4. Finished.

## hot key

Open the plugin:
- macOS:Command + option + N
- Windows: Control + alt + N

Confirm:
- macOS:Command + Enter
- Windows: Control + Enter
